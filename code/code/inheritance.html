  • Default link color.
  • Inherit. This link inherints the parents body color in this case, as <a> is a child of the <li> element, which is a child of the <body> element, set to green.
  • Initial. Resets the link color to the browsers default style sheet color for the element (in this case the <li> element). The <a> is nested inside the <li> element, and class selector (class="my-class-2") is in the <li> element.
  • Unset. Reset the link to its natural color.

With css selector "li a" valued to the color red, (a {color: red}) the above first and last list item links will become red.
