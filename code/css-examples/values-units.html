ems and rems

  • The em unit means "parent elements font size".
  • These li inside this ul with a class of ems takes its sizing from its parents.
  • Each successive level of nesting
    • gets progressively larger
    • as each has its font size set to 1.3em
      • -1.3 times its parents font size.
  • The rem unit means "The root element's font-size".
  • Each successive level of nesting does not keep getting larger.
