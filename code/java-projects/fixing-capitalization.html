Fixing Capitalization

Back to the Code Repository / MDN web docs: Fixing Capitalization
  1. Convert the whole of the string contained in the input variable to lower case and store it in a new variable (lower).
  2. Grab the first letter of the string in this new variable and store it in another variable (firstLetter).
  3. Using this latest variable (firstLetter) as a substring,
    replace the first letter of the lowercase string (lower.replace(firstLetter,) with the first letter of the lowercase string changed to upper case (firstLetter.toUpperCase())
    Store the result of this replace procedure in another new variablecapitalized.
  4. Change the value of the result variable (capitalized) to equal to the final result, not the input variable.
