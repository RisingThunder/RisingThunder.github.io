Making new strings from old parts

Back to the Code Repository / MDN web docs: Making new strings from old parts
  1. Extract the three-letter station code and store it in a new variable (stationCode).
  2. Find the character index number of the semicolon.
  3. Extract the human-readable station name using the semicolon character index number (indexOf(';')) as a reference point, and store it in a new variable (semiColon).
  4. Concatenate the two new variables (stationCode)and (stationName) and a string literal ( + ': ' + ) to make the final string (stationCode + ': ' + stationName).
    Changing the value of the result variable to equal to the final string stationCode + ': ' + stationName, not the input variable.
